# Lab 2 - Interaction

## Project Description
This lab is a web development exercise using HTML and CSS. It focuses on understanding the basics of web page structure, layout design, and styling. The lab helps students build foundational skills in creating simple web pages using standard web development technologies.

---

## SDG Theme
This project aligns with:

**SDG 4: Quality Education**

It supports learning by:
- Strengthening basic HTML and CSS skills
- Encouraging structured web development practice
- Improving understanding of web page layout and design

---

## Features
- Basic HTML page structure
- Simple CSS styling
- Clean and minimal user interface
- Organized content layout
- Beginner-friendly implementation
